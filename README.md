# Libft

Library who contain most of libc's functions

[atoi](/srcs/ft_atoi.c)
[bzero](/srcs/ft_bzero.c)
[error](/srcs/ft_error.c)
[get_next_line](/srcs/ft_get_next_line.c)

## How to

### Compile

```
$ make
```

### Clean

Delete all compilation files

```
$ make clean
```

Delete all compilations files and binarie

```
$ make fclean
```
